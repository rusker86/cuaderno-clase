Pasos para escribir un programa

  1. Planificar el desarrollo del software
  2. Descargar el entorno de desarrollo o editor de código
  3. Escribir el programa
  4. Realizar test para buscar bugs. En caso de que los haya corregirlos
  5. Ejecutar y probar el programa completo
  6. Escribir documentación (aburrido)
