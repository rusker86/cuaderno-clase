Entidades HTML — Repaso ampliado

En HTML podemos usar entidades con nombre (p.ej. &copy;) o entidades numéricas (p.ej. &#169;). Son útiles cuando un carácter se confunde con la sintaxis de HTML (como <, &) o para tipografía correcta (elipsis, guiones, símbolos).
