Entidades HTML — Repaso ampliado

Las entidades con nombre son aquellas que tienen el nombre del símbolo en su declaracíon. Mientras que las entidades numéricas usa el código ASCII del símbolo correspondiente

Categoría Ejemplo renderizado Entidad con nombre Entidad numérica Uso/Comentario
Moneda &euro &euro; Se usa para representar cantidades monetarias
Flecha → &rarr; Se usa para representar flechas
Elipsis … &hellip; Se usa para mostrar una elipsis
Derechos y marcas © &copy; Se usa para representar marcas comerciales y de derechos de autor
Espacio   &nbsp; Se usa para introducir un espacio en blanco (Se ha resaltado con mark para ver el espacio)
