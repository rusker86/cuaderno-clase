Ejercicio 04: Fondos (backgrounds) en CSS

A) Fondo color sólido

Usa background-color con un color cualquiera.

B) Imagen con repeat

Usa background-image y background-repeat: repeat.

C) Imagen sin repetir y posicionada

Colócala en la esquina inferior derecha con background-position.

D) cover vs contain

Haz que la imagen cubra el contenedor (cover) o se ajuste entera (contain).

E) Fondo fijo (attachment: fixed)

El fondo queda fijo mientras el contenido se desplaza.

(Opcional) Múltiples fondos

Combina dos imágenes en capas con background-image separado por comas.
