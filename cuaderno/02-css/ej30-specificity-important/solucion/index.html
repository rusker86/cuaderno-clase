Ejercicio 30: !important y especificidad

A) Conflicto de reglas

Soy un párrafo con varios estilos en conflicto. El inline style (purple) gana sobre los selectores.

B) !important

Esta caja es roja porque la regla con !important anula otras más específicas.

(Opcional) Mal uso

Ejemplo: * { margin:0 !important; } resetea TODO y es difícil de sobrescribir.
